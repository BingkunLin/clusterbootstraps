## Clusterbootstraps

Clusterbootstraps is a Python library for dealing with cluster bootstraps.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install clusterbootstraps.

```bash
pip install clusterbootstraps
```
## Package Homepage
